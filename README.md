### McDonald's Nutritional Diagnostic Report
**Tools:** Power BI, MySQL, DAX

- Developed an interactive Power BI dashboard analyzing nutritional profiles of 57 McDonald's menu items across 8 fast-food chains (515 total records)
- Engineered DAX measures and a custom CalorieBand categorization to segment items into healthy (≤400 kcal), moderate (401–700 kcal), and high-risk (>700 kcal) tiers
- Wrote SQL queries to extract and filter nutritional data by category, sodium levels, and calorie ranges from a MySQL database
- Identified that 74% of menu items exceed the 1,000mg sodium threshold, with average sodium per item consuming 62% of the recommended daily intake
- Delivered a 9-slide diagnostic report covering calorie distribution, competitor comparison, sodium risk analysis, and a healthy menu audit
